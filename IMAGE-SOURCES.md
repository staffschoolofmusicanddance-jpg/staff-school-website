## When Replacing With Real Images

For every real image you add, record:

- File name
- Subject shown
- Source URL
- Photographer or creator
- License type
- Date added
- Whether model or parent consent is required

Suggested table format:

| File | Subject | Source URL | License | Notes |
| --- | --- | --- | --- | --- |
| `assets/images/example.webp` | Keyboard lesson | `https://...` | Owner supplied / Unsplash / Licensed | Add consent or attribution notes here |

## Owner Reminder

- Do not hotlink images from random websites.
- Prefer owner-supplied images, royalty-free sources, or properly licensed photography.
- Update image alt text when you replace placeholders with real photos.
